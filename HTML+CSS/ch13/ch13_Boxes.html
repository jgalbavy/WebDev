Border, Margin, Padding

Border: every box has a border that seperates the edge of one box from another

Margin: Sit outside the edge of the border. You can set the width to create a gap between the borders of two adjacent boxes.

Padding: Space between the border of a box and any content contained within it. Adding padding can increase readability.
